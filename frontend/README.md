# Student Task Manager (Frontend)

## Setup
1. Copy `.env` and set `VITE_API_URL` to your backend (e.g., http://localhost:5000/api).
2. Install:

```
npm install
```

3. Run dev:

```
npm run dev
```

## Features
- Signup / Login (JWT)
- Create / Edit / Delete tasks
- Toggle complete
- Filter / Sort / Search
- Drag-and-drop ordering (client-side)
- Overdue notifications (browser)
